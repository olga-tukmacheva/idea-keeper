# 🎁 Хранитель идей

Каталог идей для подарков: фиксируйте варианты заранее, а не в последний момент.

---

## 📌 О проекте

**Хранитель идей** — это небольшой веб-сервис, который помогает сохранять и структурировать идеи подарков для разных людей.  
Подходит тем, кто любит готовиться к праздникам заранее и не хочет судорожно вспоминать идеи в последний момент.

---

## ✨ Возможности

- Добавление людей (мама, коллега, подруга и т. д.)
- Создание идей подарков с полями:
  - название
  - ссылка
  - цена
  - категория
  - комментарий
  - **статус**: идея, запланировано, заказано, куплено, упаковано, подарено
- Редактирование и удаление идей
- Фильтрация:
  - по человеку
  - по статусу
  - по цене
- Автоматическая загрузка категорий из базы Supabase
- Хранение всех данных в Supabase

## 🚀 Установка и запуск

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/olga-tukmacheva/idea-keeper.git
```

### 2. Откройте проект

Просто откройте **index.html** в браузере.  
Сборка или сервер — **не требуются**.

---

## 🔑 Настройка Supabase

Откройте файл **script.js** и вставьте свои ключи:

```js
const SUPABASE_URL = "https://________.supabase.co";
const SUPABASE_KEY = "public-anon-key";
```

## 🗄 Таблицы в Supabase

### 1. people
| Поле     | Тип     | Описание          |
|----------|---------|-------------------|
| id       | uuid PK | идентификатор     |
| name     | text    | имя               |
| relation | text    | родство/связь     |

### 2. ideas
| Поле       | Тип        | Описание                 |
|------------|------------|--------------------------|
| id         | uuid PK    | идентификатор            |
| person_id  | uuid FK    | связь с человеком        |
| title      | text       | название подарка         |
| link       | text       | ссылка                   |
| price      | numeric    | цена                     |
| category   | text       | категория                |
| comment    | text       | комментарий              |
| status     | text       | статус идеи              |
| created_at | timestamptz| дата создания            |

### 3. categories
| Поле       | Тип        | Описание      |
|------------|------------|---------------|
| id         | uuid PK    | идентификатор |
| title      | text       | название      |
